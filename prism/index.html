Hello PRISM!

PRISM はエディタのようなビジュアルでコードをWeb表示するためのデザインテンプレート。 CSSとJSをダウンロード 、もしくはCDNで利用可能。

■ PRISMの表示例

        
            <h1>Hello PRISM!</h1>

■ サンプルコード

        
            
            <!DOCTYPE html>
            <html lang="ja">
            <head>
                <meta charset="UTF-8">
            
                <link rel="stylesheet" href="css/prism.css">
                <script src="js/prism.js"></script>
            
            </head>
            <body>
                <h1>Prism Template Code</h1>
                <pre class="line-numbers">
                    <code class="language-html">
                        &lt;h1&gt;Hello PRISM!&lt;/h1&gt;
                    </code>
                </pre>
            </body>
            </html>
この設定のダウンロードリンク

■ 基本型

表示例

        
            最もシンプルなコード表示設定

コード

        
            
            <pre>
                <code class="language-none">
                    最もシンプルなコード表示設定
                </code>
            </pre>

言語設定

<code>にclass=""としてプログラム言語を設定

language-none language-shell language-html language-css language-js language-php language-python language-git language-yaml

■ Line Highlight

任意の行を強調表示する

表示例

        
            ハイライトあり
            ハイライトなし

コード

        
            <pre data-line="1">
                <code class="language-html">
                    ハイライトあり
                    ハイライトなし
                </code>
            </pre>

方法

<pre>にdata-line=""としてハイライトする行を指定

1 1, 3 1, 3-5

【PRISM公式ガイド】Line Highlight: https://prismjs.com/plugins/line-highlight/

■ Line Numbers

行番号を左側に表示する

表示例

        
            行番号表示1
            行番号表示2

コード

        
            <pre class="line-numbers">
                <code class="language-html">
                    行番号表示1
                    行番号表示2
                </code>
            </pre>

方法

<pre>のclassにline-numbersを指定

【PRISM】Line Numbers: https://prismjs.com/plugins/line-numbers/

■ Command Line

コマンドライン表示する

① 表示例. デフォルト

        
            cd
            ls -a

① コード. デフォルト

        
            <pre class="command-line">
                <code class="language-shell">
                    cd
                    ls -a
                </code>
            </pre>

② 表示例. プロンプト変更・実行結果表示

        
            ls -a
            .ssh .venv
            source ~/.venv/myenv/bin/activate

② コード. プロンプト変更・実行結果表示

        
            <pre class="command-line" data-user="root" data-host="127.0.0.1" data-output="2">
                <code class="language-shell">
                    ls -a
                    .ssh .venv
                    source ~/.venv/myenv/bin/activate
                </code>
            </pre>

③ 表示例. プロンプト変更

        
            pip install mkdocs

③ コード. プロンプト変更

        
            <pre class="command-line" data-prompt="(myenv) C:\Users\watson>">
                <code class="language-shell">
                    pip install mkdocs
                </code>
            </pre>

【PRISM】Command Line: https://prismjs.com/plugins/command-line/
